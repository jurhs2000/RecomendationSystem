Sistema de Recomendación de Recetas

Sistema utilizando el lenguaje Python, con el framework Flask y Neo4j como base de datos basada en grafos.

Este sistema tiene como propósito resolver la problemática principal de recomendar recetas,basadas en los ingredientes que posee un usuario, mostrando así, recetas que puedes cocinar y el porcentaje de ingredientes que tienes de esa receeta. Además puedes filtrar según el sabor, tipo de comida o método de preparación, para obtener recomendaciones a tu gusto.
